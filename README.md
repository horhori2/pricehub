# PriceHub

트레이딩카드(포켓몬 / 원피스 / 디지몬) 시세를 매일 자동으로 수집하고,
그 데이터를 바탕으로 판매가를 관리하는 내부 대시보드 + API 서버입니다.

## 지원 카테고리

| 게임 | 한글판 | 일본판 |
|---|---|---|
| 포켓몬 | ✅ | ✅ |
| 원피스 | ✅ | - |
| 디지몬 | ✅ | - |

## 주요 기능

### 가격 자동 수집
- 네이버 쇼핑 API로 한글판 카드(포켓몬/원피스/디지몬) 시세를 매일 자동 수집.
- 일본판 포켓몬 카드는 카드러쉬(cardrush-pokemon.jp)·유유테이 사이트를 크롤링해 수집.
- 레어도·미러·특일 등 카드별 조건을 반영한 검색어 생성 및 상품명 필터링으로 오매칭 방지.
- 수집 결과는 카드별 가격 히스토리로 누적 저장.

### 판매가 관리 대시보드
- 확장팩별 카드 목록에서 개별 판매가 설정, 정렬(판매가/시장 최저가), 레어도·즐겨찾기 필터.
- 엑셀 업로드 기반 일괄 판매가 설정 — 판매처 우선순위 지정, 신규/유지/상승/하락 케이스별 자동 분류.
- 판매가 상승·하락 대기 목록에서 관리자가 검수 후 확정 반영 (레어도 오매칭 등 확인용 안전장치).
- 판매가 미설정 카드 목록.

### 저가 경고
- 판매가가 시장 최저가보다 낮은 카드를 자동으로 감지해 목록/배너로 표시.
- 시장 최저가는 자체 매장(화성스토어-TCG- 등) 항목을 제외한 경쟁사 기준으로 계산.

### 판매처 통계
- 확장팩 단위로 판매처(쇼핑몰)별 등록 건수·평균가 랭킹 집계.

### 매입리스트 관리
- 게임별 매입리스트를 만들어 카드를 담고, 판매가 대비 매입가 비율(기본 50%)로 추천 매입가 자동 계산.
- 가격 관리자가 추천가를 참고해 최종 매입가 확정.

### 외부 연동 REST API
- API Key 인증(`Authorization: Api-Key <key>`) 기반 개발자용 API.
- 확장팩/카드 목록·검색, 카드 상세 + 가격 이력, 상품코드 조회, 수집 현황 요약, 매입리스트 조회 제공.
- API Key당 요청 빈도 제한(rate limiting) 적용.

### 기타
- 즐겨찾기, 카드 검색, 판매가 일괄/개별 초기화.

## 기술 스택

- Django 5 + Django REST Framework
- MySQL
- 네이버 쇼핑 API, BeautifulSoup(웹 스크래핑)
