개요
PriceHub API는 포켓몬 카드 & 원피스 카드의 판매가(selling_price)와 시장가 데이터를 제공합니다.
외부 프로그램(엑셀 도구, 재고 관리 시스템 등)에서 판매가를 조회하거나 카드 정보를 검색할 때 사용하세요.

Base URL

http://your-server/api/pokemon/kr/ # 포켓몬 한글판 http://your-server/api/pokemon/jp/ # 포켓몬 일본판 http://your-server/api/onepiece/kr/ # 원피스 한글판 http://your-server/api/digimon/kr/ # 디지몬 한글판

응답 형식

모든 응답은 JSON 형식입니다. 목록 API는 페이지네이션을 포함합니다.
{ "count": 100, "next": "http://your-server/api/pokemon/kr/cards/search/?page=2", "previous": null, "results": [ ... ] }
인증
모든 API 요청에는 Api-Key 인증이 필요합니다. API 키는 관리자에게 문의하세요.

요청 헤더

Authorization: Api-Key YOUR_API_KEY Accept: application/json

Python 예시

import urllib.request, json API_KEY = "your-api-key" BASE_URL = "http://your-server/api/pokemon/kr" req = urllib.request.Request(f"{BASE_URL}/cards/by-product-code/PKM-M2A-001-K/") req.add_header("Authorization", f"Api-Key {API_KEY}") req.add_header("Accept", "application/json") with urllib.request.urlopen(req) as resp: data = json.loads(resp.read().decode()) print(data['selling_price'])
⚠️ API 키는 외부에 노출되지 않도록 주의하세요. 환경변수나 별도 설정 파일로 관리하는 것을 권장합니다.
포켓몬 한글판 — 상품코드로 카드 조회
네이버 스토어 판매자 상품코드로 카드를 조회합니다. 엑셀 가격 업데이트 도구에서 사용하는 핵심 API입니다.
GET /api/pokemon/kr/cards/by-product-code/{shop_product_code}/ ▶
🔑 Api-Key 인증 필요
Path Parameters
파라미터타입필수설명
shop_product_codestring필수네이버 판매자 상품코드 (예: PKM-M2A-001-K)
요청 예시
GET /api/pokemon/kr/cards/by-product-code/PKM-M2A-001-K/ Authorization: Api-Key YOUR_API_KEY
응답 예시 (200 OK)
{ "id": 7655, "card_number": "001", "name": "뿔충이", "rarity": "C", "selling_price": 200, "shop_product_code": "PKM-M2A-001-K", "expansion": { "code": "M2A", "name": "MEGA드림ex" } }
응답 필드
필드타입설명
idinteger카드 고유 ID
card_numberstring카드 번호 (예: 001)
namestring카드명
raritystring레어도 (C, R, RR, SAR 등)
selling_priceinteger | null설정된 판매가 (미설정 시 null)
shop_product_codestring네이버 판매자 상품코드
expansionobject확장팩 정보 (code, name)
오류 응답
# 404 — 카드 없음 { "error": "상품코드 'PKM-XXX' 에 해당하는 카드가 없습니다." } # 401 — 인증 실패 { "detail": "Authentication credentials were not provided." }

📊 엑셀 자동 업데이트 도구
update_prices_from_server.py 스크립트를 사용하면 엑셀 파일의 판매자 상품코드(B열)로 서버에서 판매가를 조회해 자동으로 업데이트합니다.

작동 방식

엑셀 B열 (판매자 상품코드) ↓ GET /api/pokemon/kr/cards/by-product-code/{code}/ ↓ selling_price 반환 ↓ F열 가격 업데이트 + 색상 표시 ↓ {파일명}_수정.xlsx 저장

설정 방법

SERVER_URL = "http://your-server-address" # 서버 주소 API_KEY = "your-api-key" # API 키 PRICE_COL = 6 # F열 (가격이 있는 열) CODE_COL = 2 # B열 (상품코드가 있는 열) MAX_WORKERS = 15 # 동시 요청 수 (로컬: 30까지 가능, 원격: 10~15 권장)

색상의미
흰색판매가 동일 (변경 없음)
